How to Run the Simulator

Run this command from the project root. Events are emitted as JSON to stdout. Use the POST hook to ingest events:

# Basic run (uses rate from DB config)
python simulator/simulator.py

# With custom rate
SIM_RATE=5 python simulator/simulator.py

# Pipe events into the API endpoint
python simulator/simulator.py | while read line; do
  curl -s -X POST http://localhost:5000/api/simulator/event \
    -H "Content-Type: application/json" -d "$line";
done
